AI-powered healthcare system for early detection of neurological disorders using machine learning and full-stack web technologies.
# NeuroCare

AI-based system for detecting neurological disorders like Alzheimer’s, Parkinson’s, and Multiple Sclerosis.

## Tech Stack
- React (Frontend)
- Flask (Backend)
- Machine Learning

## Features
- Disease prediction using AI
- Doctor & patient dashboards
- Case tracking system

## Setup
1. Install backend requirements
2. Run Flask server
3. Run frontend using npm

## Note
ML models and datasets are not included due to size limits.
