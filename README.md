AI-powered healthcare system for early detection of neurological disorders using machine learning and full-stack web technologies.

---

## 📌 Overview

NeuroCare is a full-stack web application that helps in detecting neurological diseases such as Alzheimer’s, Parkinson’s, and Multiple Sclerosis using AI models.  
It provides a platform for patients and doctors to manage cases, view results, and track progress.

---

## 🚀 Features

- 🧠 Alzheimer Detection (MRI Images)
- 🎤 Parkinson Detection (Voice Analysis)
- 🧬 Multiple Sclerosis Detection
- 👨‍⚕️ Doctor Dashboard
- 🧑‍💻 Patient Dashboard
- 📊 Case Tracking & Reports

---

## 🛠️ Tech Stack

### Frontend
- React.js

### Backend
- Python (Flask)
- REST APIs

### Machine Learning
- TensorFlow / Keras
- Scikit-learn
